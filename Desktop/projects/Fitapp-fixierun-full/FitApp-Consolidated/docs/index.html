API Documentation

The API is built using FastAPI and follows RESTful principles.

Endpoints

GET /api/v1/auth/login - Login endpoint
POST /api/v1/auth/register - Registration endpoint
GET /api/v1/activity - Get activity data
POST /api/v1/activity - Create new activity
GET /api/v1/nft - Get NFT data
POST /api/v1/nft/mint - Mint new NFT

Blockchain Integration

FitApp integrates with the Ethereum blockchain for NFT minting and token transfers.

Integration Points

// NFT Minting
async function mintNFT(activityData) {
const contract = new ethers.Contract(contractAddress, abi, signer);
const tx = await contract.mintNFT(activityData);
await tx.wait();
}

// Token Transfers
async function transferTokens(toAddress, amount) {
const contract = new ethers.Contract(tokenAddress, tokenAbi, signer);
const tx = await contract.transfer(toAddress, amount);
await tx.wait();
}
